Version 0.1.1 Changelog (07/03/2019)

Note: None of these changes have been balanced at all, so any suggestions are welcome.

  • Rewrote the entire Game to make it dynamic instead of having to write seperate functions for each building, staff and upgrade type.
  • Added more building types. Office Buildings, Tower Blocks, Skyscrapers and Complexes.
  • Added mouse click upgrades. After a certain amount of clicks, an upgrade will pop that allows you to increase how much money is earned when manually clicking.
  • Added variable "inflation" rates for each staff type.
  • 5 Upgrades available for Apprentices
  • 5 Upgrades available for Mouse Clicking
  • 1 Upgrade available for Full-Timers
